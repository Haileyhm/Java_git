*상속
: 상속을 통해서 여러개의 클래스를 공통으로 사용할 수 있음

사용법
public class 자식클래스 extends 부모클래스 : 부모클래스를 확장하여 자식클래스 만듦

목적
교수님 정리
1) 기능추가(확장)
2) 부모클래스 이름으로 변수지정
3) 매서드 강제를 위해서_ 협업 시 특정 매서드만 사용하도록

======
1) 여러 클래스를 하나의 공통된 형태로 사용하기 위해 사용
2) 미리 정한 특성을 공유하기 위해 사용
   ex.특정 변수, 생성자,  
       String name = "";
       this.name = ""

-부모클래스에 있는 매서드가 자식클래스에 없으면 부모클래스의 매서드 사용
 부모클래스에 있는 매서드가 자식클래스에 있으면 자식클래스의 매서드 사용 

-상속 이뤄졌을 때 부모클래스 접근
: 자식클래스에서 동일한 이름의 부모클래스의 매서드 실행하고 싶으면, 
super.매서드이름();

-확장해서 자식클래스 만들없는데 내용이 아무것도 없으면 부모클래스랑 완전하게 같은 클래스가 됨.

- 메인클래스에 클래스의 변수 명명할 때

1)부모클래스 변수명 = new 자식클래스();

2)부모클래스 변수명 = new 부모클래스();
3)자식클래스 변수명 = new 자식클래스();

앞의 클래스는 변수를 구분하기 위해 넣어줌 -> 부모클래스로 구분할 수도 있음 
new 뒤의 클래스는 실제 하는일  -> 원하는 매소드가 자식클래스에 있다면 앞의 클래스도 자식클래스로 써줘야해

굳이, 1) 처럼 사용하는 이유는 특정한 분류를 통해 묶어야 할 필요성이 있을 때 
	실제 하는 일은 자식클래스의 매서드야
